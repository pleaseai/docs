# Components Overview

The layer provides several pre-built components for documentation sites, organized into three categories: Layout, Content/MDC, and UI components.

## Layout Components

### AppHeader

The site header with navigation and dark mode toggle.

```vue
<AppHeader />
```

Features:
- Site title/logo
- Navigation links
- Dark/light mode toggle
- Mobile responsive

### AppFooter

The site footer.

```vue
<AppFooter />
```

### DocsSidebar

Navigation sidebar for documentation pages.

```vue
<DocsSidebar />
```

Features:
- Hierarchical navigation
- Active state highlighting
- Collapsible sections

### DocsTableOfContents

Table of contents for the current page.

```vue
<DocsTableOfContents :toc="toc" />
```

Features:
- Auto-generated from headings
- Scroll-aware active state
- Depth-based indentation

### DocsPageHeader

Page header with title and description.

```vue
<DocsPageHeader
  title="Page Title"
  description="Page description"
/>
```

### DocsPageNav

Previous/Next navigation.

```vue
<DocsPageNav
  :prev="{ path: '/prev', title: 'Previous' }"
  :next="{ path: '/next', title: 'Next' }"
/>
```

## Content Components (MDC)

These components are globally registered and can be used directly in Markdown files using MDC syntax.

| Component | Description |
|-----------|-------------|
| [Callout](/docs/components/callout) | Alert and notification boxes |
| [Steps](/docs/components/steps) | Step-by-step guides |
| [Tabs](/docs/components/tabs) | Tabbed content sections |
| [CodeTabs](/docs/components/code-tabs) | Package manager code switcher |
| [ComponentPreview](/docs/components/component-preview) | Component demo previews |
| [LinkedCard](/docs/components/linked-card) | Card-style navigation links |
| [CodeCollapsibleWrapper](/docs/components/code-collapsible) | Collapsible code blocks |
| [CopyButton](/docs/components/copy-button) | Copy to clipboard button |
| [ButtonA](/docs/components/button-a) | Button-styled links |
| [Prose](/docs/components/prose) | Markdown element styling |

## UI Components

The layer includes shadcn-vue components:

### Button

```vue
<Button variant="default">Click me</Button>
<Button variant="outline">Outline</Button>
<Button variant="ghost">Ghost</Button>
```

### Separator

```vue
<Separator />
<Separator orientation="vertical" />
```

## Customization

All components can be overridden by creating a component with the same name in your project's `components` directory.
