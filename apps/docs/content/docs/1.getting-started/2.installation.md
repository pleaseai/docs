# Installation

Get started with `@pleaseai/docs-layer` in your Nuxt project.

## Prerequisites

- Node.js 18+
- Nuxt 4.x
- pnpm, npm, or bun

## Install the Layer

```bash
# Using pnpm
pnpm add @pleaseai/docs-layer

# Using npm
npm install @pleaseai/docs-layer

# Using bun
bun add @pleaseai/docs-layer
```

## Configure Nuxt

Extend the layer in your `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
  extends: ['@pleaseai/docs-layer'],

  compatibilityDate: '2025-01-01',
})
```

## Create Content

Create a `content` directory and add your first page:

```md
## <!-- content/index.md -->

title: Welcome
description: My documentation site

---

# Welcome to My Docs

This is my documentation site.
```

## Run Development Server

```bash
pnpm dev
```

Your documentation site is now running at `http://localhost:3000`.
