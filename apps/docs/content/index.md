# Docs Please

A modern documentation system built with **shadcn-vue** and **Nuxt Content**.

## Features

- **shadcn-vue Components** - Beautiful, accessible UI components
- **Tailwind CSS v4** - Modern CSS with CSS variables
- **Dark Mode** - Built-in dark/light mode support
- **Mobile Responsive** - Works on all screen sizes
- **Markdown Support** - Write documentation in Markdown

## Quick Start

```bash
# Install the layer
pnpm add @pleaseai/docs

# Extend in nuxt.config.ts
export default defineNuxtConfig({
  extends: ['@pleaseai/docs']
})
```
